## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/olgaAsmith/circle.git
   ```
2. Перейдите в папку проекта
   ```bash
   cd circle
   ```
3. Установите зависимости:

   ```bash
   npm install
   ```

## Запуск проекта

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

### Сборка для production

```bash
npm run build
```
